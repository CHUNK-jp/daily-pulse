✨Features

🔒

100% Private

Run with local Ollama — nothing leaves your machine. No accounts, no telemetry.

📡

Multi-Source

HackerNews, any RSS/Atom feed, and Reddit — all in one brief.

🤖

AI Summarized

Powered by Ollama (local) or Claude API. Picks the top 3–5 stories and explains why they matter.

🌐

EN / JA

Output in English or Japanese. One flag: --lang ja.

⏰

Automatable

Works with cron, GitHub Actions, launchd — set it and forget it.

📄

Markdown or HTML

Output to stdout, a file, or pipe into your note-taking app.

🚀Quick Start

# 1. Clone
git clone https://github.com/CHUNK-jp/daily-pulse.git
cd daily-pulse

# 2. Install dependencies
pip install -r requirements.txt

# 3a. Run with local LLM (free)
ollama pull llama3.2
python3 daily_pulse.py run

# 3b. Or use Claude API
ANTHROPIC_API_KEY=sk-... python3 daily_pulse.py run --provider claude

# Save to file
python3 daily_pulse.py run -o ~/brief.md

# Japanese output
python3 daily_pulse.py run --lang ja

📖Commands

CommandDescription
runFetch all sources and print AI brief
run -o FILESave brief to a file
run --format htmlOutput as HTML instead of Markdown
run --lang jaSummarize in Japanese
run --provider claudeUse Claude API instead of Ollama
initCreate default config at ~/.config/daily-pulse/config.yaml
sourcesList all configured sources
