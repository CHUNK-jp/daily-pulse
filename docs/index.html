FEATURES

これだけで十分。

必要なものだけ。余計なもは一切なし。

📡

複数ソースを自動収集

HackerNews・RSS・Redditを一括取得。設定ファイルで自由にカスタマイズ可能。

🧠

ローカルLLMで要約

Ollamaを使えばすべてがローカルで完結。APIキー不要、課金なし、トラッキングなし。

🌐

日本語出力対応

--lang ja を付けるだけで日本語のブリーフを生成。英語にも対応。

📄

Markdown / HTML 出力

ファイルへの保存、HTMLレポート出力にも対応。cronで毎朝自動実行も簡単。

☁️

Claude APIにも対応

--provider claude でAnthropicのAPIに切り替え。モデルも設定ファイルで変更可。

⚙️

YAMLで柔軟設定

ソース・AI・出力形式をすべてYAMLで管理。init コマンドで即セットc��ップ。

QUICK START

すぐ使える。

5分以内に動きます。

1

リポジトリをクローン

git clone https://github.com/CHUNK-jp/daily-pulse.git
cd daily-pulse
2

依存ライブラリをインストール

pip install -r requirements.txt
3

設定ファイルを作成

python3 daily_pulse.py init

または config.example.yaml を ~/.config/daily-pulse/config.yaml にコピーして編集。

4

Ollamaをセットc��ップ（ローカルLLM使用時）

ollama pull llama3.2
5

実行！

# ターミナルに表示
python3 daily_pulse.py run

# 日本語でファイルに保存
python3 daily_pulse.py run --lang ja -o ~/brief.md

# Claude APIを使用
ANTHROPIC_API_KEY=sk-... python3 daily_pulse.py run --provider claude
