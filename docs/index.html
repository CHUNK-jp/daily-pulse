🚀 Quick Start

# Clone & install
git clone https://github.com/CHUNK-jp/daily-pulse.git && cd daily-pulse
pip install -r requirements.txt

# Local LLM (free)
ollama pull llama3.2
python3 daily_pulse.py run

# Claude API
ANTHROPIC_API_KEY=sk-... python3 daily_pulse.py run --provider claude

# Save / Japanese
python3 daily_pulse.py run -o ~/brief.md --lang ja

📖 Commands

CommandDescription

run --lang jaSummarize in Japanese
